Default Apps Page

This module changes the default home page to show all available apps after login.

Features

Apps Menu as Home Page

Users will see the Apps menu immediately after login instead of dashboard.

Easy Access

Quickly browse and install apps without navigating through menus.

Clean Interface

Simple and straightforward interface for better user experience.

Installation

  1. Install the module
  2. Logout and login again
  3. You'll see the Apps page as your home page
